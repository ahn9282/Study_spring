VI 편집기

복사 : yy
붙이기 : p
저장 : wq, ZZ
모드 전환 : i, a, o, ESC

i는 그냥 입력모드
o는 개행 후 입력모드
a는 현재 커서 바로 앞에서 입력모드
I 현재 행의 첫 컬럼으로 이동
O 현재 행의 이전 행에 입력모드

ctrl + u  : 화면의 절반 만큼 위로 이동
ctrl + d : 화면의 절반만큼 아래로 이동
ctrl + b : 화면 하나 만큼 위로
ctrl  + f : 화면 하나만큼 아래로
ctrl + y : 화면을 한행 위로
ctrl + e : 화면을 한행 아래로
shift + g : 맨아래 행으로 이동
: + 숫자 : 해당 숫자의 행으로 이동
:  + $ : 파일의 마지막 행으로 커서 이동
명령모드 r : 한키만 수정할 때 사용하는 명령키
명령모드 cw : 해당 커서가 속한 단어위치에서 커서부터 단어앞까지 지우고 입력모드 ex) 대충 단어 중간에서 ctrl + shift ←
명령 모드 cc : 한줄 삭제
명령모드 u : 이전 명령전으로 되돌리기
:set number, :set nu : 행번호로 보여지도록 하는 명령어
명령모드 x : 한글자만 삭제 입력모드로 전환x
명령모드 dw : 단어 삭제 입력모드로 전환x
명령모드 dd : 한 줄 삭제 입력모드로 전환x
shift + d : 한줄 삭제이며 행은 유지
